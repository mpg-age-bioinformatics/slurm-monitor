# slurmOnitor

## About

a slurm cluster load monitor

## Usage

Check `slurm` variable (`False` for ssh queries). 
Run `./slurmOnitor` to start.

## Issues

Issue when using module to change python path. Solved by the following:

`export PYTHONUSERBASE="$HOME/.local"`
